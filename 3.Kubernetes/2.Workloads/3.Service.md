## Services

El Service es un objeto cuyo objetivo es agrupar un conjunto de extremos de Pod en un solo recurso. Puedes configurar varias formas para acceder a la agrupaci√≥n.

### Ejemplo

Crear un archivo [3.service.yaml](3.service.yaml)
```bash
apiVersion: v1
kind: Service
metadata:
  name: my-service
spec:
  selector:
    app.kubernetes.io/name: nginx-alpine-deployment
  ports:
    - protocol: TCP
      port: 80
      targetPort: 80
```

Para verificar:

```bash
kubectl apply -f 3.service.yaml
```

```bash
kubectl get service
```
